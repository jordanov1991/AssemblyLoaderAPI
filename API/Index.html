Call dynamically loaded assembly method

Call Instructions


Service takes a json request at /api/execute using an HttpPost method.
The Body of the method must contain a json with the following structure:

{
"dlllocation" : "\\some_location\\some_assembly.dll",
"dllnamespace": "DummyNamespace",
"dllclass": "DummyClassName",
"dllmethod": "DummyMethodName"
}
